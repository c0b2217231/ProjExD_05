# ゲーム のタイトル

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクター豪華豚（ゴウカトン ）をマウス操作により丸焼きにするゲームで，・・・

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
* 敵機に関するクラス
* 追加機能2に関するクラス
### 担当追加機能
* ハイスコアが更新されたときに表示
